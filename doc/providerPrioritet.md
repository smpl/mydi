# Пример демонстрирующий приоритет провайдеров

Порядок элементов переданных в конструктор **Container** важен, когда Container не может найти определение, 
он начинает искать **Provider** который бы смог загрузить нужный ему контейнер, для этого он в цикле проходит по всем 
провайдерам и вызывает у них метод has, первый provider который возвращает true и используется.

Таким образом можно например публиковать в git репозитории параметры по умолчанию, а реальные параметры не 
публиковать, добавив их в .gitignore например строку private.* и подключать провайдеры в случае если файлы существуют.
